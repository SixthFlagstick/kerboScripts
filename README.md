# kerboScripts
A collection of all of the scripts I use in my modded KSP career save, with the goal of eventually having full automation of every mission using only kOS.

### Mod Requirements
My scripts are designed to interact with the following mods - they may function differently (or not at all) without them.
 - Ferram Aerospace Research
 - RealChute
 - Principia
 - RemoteTech
 - Kerbalism (Science-Only)

### Disclaimer
This repository is primarily serves as an online backup/archive of my scripts. 
You are welcome to download and use them, but some scripts may require edits to work correctly with your crafts or setup.
